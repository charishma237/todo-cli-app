# 📝 CLI To-Do App (Python)

A simple command-line To-Do List app built using Python and Object-Oriented Programming.

## 🚀 Features

- Add tasks ✅
- View all tasks 🗂
- Mark as done 🎯
- Delete tasks 🗑
- CLI-based interaction

## 🧠 Concepts Used

- Classes and Objects
- Exception Handling
- List and Dictionary operations
- User Input and File I/O

## 📂 How to Run 

---
python todo.py
---
